Scientific Calculator
​A Python-based scientific calculator that handles advanced math like trigonometry, logs, and roots using the built-in math library. It features a clean CLI, robust error handling, and precise floating-point logic.
​✨ Features
​This calculator runs entirely in the terminal with an easy-to-use numbered menu. It includes over 30 mathematical operations across several categories:
​Basic Arithmetic: Addition, Subtraction, Multiplication, Division, Modulus, and Power.
​Roots & Exponents: Square roots, Cube roots, Nth roots, and Exponentials (e^x).
​Logarithms: Natural Log (ln), Base-10 Log, and custom base logarithms.
​Trigonometry: Sine, Cosine, Tangent (inputs in degrees), their inverse functions (arcsin, arccos, arctan), and hyperbolic functions.
​Combinatorics & Number Theory: Factorials (n!), Permutations P(n,r), Combinations C(n,r), Greatest Common Divisor (GCD), and Least Common Multiple (LCM).
​Miscellaneous Operations: Absolute values, Floor, Ceiling, Rounding, and quick access to constants (π, e, φ, √2, √3).
​🚀 Getting Started
​Prerequisites
​Python 3.x installed on your system.
​No external libraries or dependencies are required (relies strictly on Python's standard math module).
1. Clone this repository to your local machine:
git clone https://github.com/Yuvraj-glitch1/Scientific-calculator-basic-.py-code-/edit/main/README.md
2. Navigate to the project directory:
cd Scientific-calculator-basic-.py-code-
3. Run the script:
python calculator.py
​🛡️ Error Handling
​The script features built-in error handling to prevent crashes from common mathematical and input errors, including:
​Division by zero.
​Square roots of negative numbers.
​Undefined logarithms (e.g., zero or negative inputs).
​Non-numeric string inputs (catches ValueError).
